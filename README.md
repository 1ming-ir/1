# 台股多標的技術策略分析

## 專題概述

本專題以台股 KBar 資料為基礎，建立一個可在網頁上操作的多標的策略回測系統。分析標的包含 0050、2330 與 2357，分別代表 ETF、半導體權值股與電子品牌股。透過相同的策略規則套用在不同標的上，可以觀察策略表現是否具有一致性，也能比較不同標的在報酬與風險上的差異。

系統除了提供單一標的的策略績效比較，也加入跨標的比較功能。例如可以將同一個 MACD、KDJ 或布林通道策略，同時套用在 0050、2330、2357 上，觀察哪個標的在相同交易邏輯下較穩定。

## 資料來源

- 資料庫：`shioaji.db`
- 使用資料表：`stock_KBar_0050`、`stock_KBar_2330`、`stock_KBar_2357`
- 分析標的：
  - 0050 元大台灣50
  - 2330 台積電
  - 2357 華碩
- 分析期間：2023-04-17 至 2025-03-10
- 部署資料檔：`data/stock_pool_0050_2330_2357_2023_2025.csv.gz`

## 系統功能

- 可選擇主要分析標的。
- 可切換 KBar 週期：5 分、15 分、30 分、60 分、日 K。
- 可設定交易數量與停損點數。
- 可比較主要標的下不同策略的績效。
- 可將同一策略套用到 0050、2330、2357 進行跨標的比較。
- 可顯示價格走勢、權益曲線、交易明細與策略排行榜。
- 可進行主要標的參數搜尋。
- 可下載績效表、跨標的比較表、交易紀錄與最佳參數彙整。
- 可串接 DeepSeek API 進行策略評析與即時追問。

## 策略內容

| 策略 | 說明 |
| --- | --- |
| 均線趨勢 | 透過短期與長期移動平均線交叉判斷趨勢方向。 |
| RSI 動能 | 比較短週期 RSI 與長週期 RSI，用來觀察動能轉強或轉弱。 |
| RSI 反轉 | 使用 RSI 超買超賣區間尋找反彈或回落訊號。 |
| 布林突破 | 以布林通道上下軌判斷價格突破與回歸。 |
| MACD 交叉 | 使用 DIF 與 DEA 的黃金交叉、死亡交叉產生交易訊號，並加入零軸方向過濾。 |
| KDJ 轉折 | 透過 K、D 線交叉與超買超賣區間判斷短線轉折。 |

## 績效評估

策略績效不只看淨損益，也加入風險與穩定性指標：

- 交易次數
- 勝率
- 平均每筆損益
- 最大回撤
- Profit Factor
- 報酬回撤比
- 穩定性指標
- 每筆交易期望值
- 綜合評分

綜合評分會同時考慮報酬與風險，避免只因為單一策略在某段期間損益較高，就忽略回撤過大的問題。

## 跨標的比較設計

本系統特別加入跨標的比較功能。使用者可以指定一個策略，系統會將該策略分別套用在 0050、2330 與 2357，並產生同一張績效比較表。這個設計可以觀察：

- 策略是否只適合單一個股。
- ETF 與個股在相同策略下的風險差異。
- 哪個標的在相同策略下報酬回撤比較佳。
- 策略是否容易受到個股波動特性影響。

## 參數搜尋

系統可針對主要分析標的進行參數搜尋，例如：

- 均線快慢週期
- RSI 期間與上下界
- 布林通道期間與標準差倍數
- MACD 快線、慢線與訊號線期間
- KDJ RSV、K、D 平滑期間

搜尋結果依綜合評分排序，並整理各策略最佳設定。

## AI 分析

系統可串接 DeepSeek API，根據主要標的策略績效、跨標的比較結果與參數搜尋摘要，自動整理策略優缺點、風險特徵與改善方向。若未設定 API key，系統仍會提供基本的離線摘要。
