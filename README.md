# 2330 技術策略研究室

## 專題概述

本專題使用台積電 2330 的 KBar 歷史資料，建立一個可以在網頁上操作的技術策略回測系統。系統可調整資料區間、KBar 週期、交易數量與停損條件，並比較多種常見技術指標策略的績效表現。

## 資料與分析標的

- 資料來源：`shioaji.db`
- 使用資料表：`stock_KBar_2330`
- 分析標的：台積電 2330
- 分析期間：2022-01-01 至 2024-04-09
- 部署資料檔：`data/stock_KBar_2330_2022_2024.csv.gz`

## 策略內容

系統包含六種策略：

| 策略 | 說明 |
| --- | --- |
| 均線趨勢 | 透過短期與長期移動平均線交叉判斷趨勢方向。 |
| RSI 動能 | 比較短週期 RSI 與長週期 RSI，用來觀察動能轉強或轉弱。 |
| RSI 反轉 | 使用 RSI 超買超賣區間尋找反彈或回落訊號。 |
| 布林突破 | 以布林通道上下軌判斷價格突破與回歸。 |
| MACD 交叉 | 使用 DIF 與 DEA 的黃金交叉、死亡交叉產生交易訊號。 |
| KDJ 轉折 | 透過 K、D 線交叉與超買超賣區間判斷短線轉折。 |

## 績效評估

策略比較不只觀察淨損益，也加入下列風險與穩定性指標：

- 交易次數
- 勝率
- 平均每筆損益
- 最大回撤
- Profit Factor
- 報酬回撤比
- 穩定性指標
- 每筆交易期望值
- 綜合評分

綜合評分會同時考慮報酬與風險，用來做策略排名與參數搜尋排序。

## 參數搜尋

系統可針對不同策略進行參數搜尋，例如均線週期、RSI 期間、布林通道標準差倍數、MACD 快慢線期間與 KDJ 平滑期間。搜尋結果會依綜合評分排序，並整理各策略最佳設定。

## AI 分析

系統可串接 DeepSeek API，根據目前回測結果與參數搜尋摘要，自動整理策略優缺點與風險提醒。若未設定 API key，系統仍會提供基本的離線摘要。

## 執行方式

```bash
pip install -r requirements.txt
streamlit run main.py
```

Streamlit Cloud 部署時，主程式請設定為：

```text
main.py
```

若要啟用 DeepSeek 分析，在 Streamlit Cloud Secrets 加入：

```toml
DEEPSEEK_API_KEY = "自己的 DeepSeek API key"
```

## 注意事項

本系統為課程作業與歷史資料分析用途。回測結果不代表未來績效，實際交易仍須考慮交易成本、滑價、流動性與市場風險。
